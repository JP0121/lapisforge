# 🛠️ Repair Shop Command Center

A sleek, modern administrative dashboard built for managing repair shop operations. This application handles live scheduling and general shop management, all wrapped in a highly responsive, dark-themed UI.

## ✨ Features

* **🔒 Secure Admin Access:** Session-based authentication with a stylized passcode entry system.
* **📅 Live Calendar Scheduling:** Interactive calendar to set and manage available time slots. Dots indicate dates with active availability.
* **⚡ Real-Time Database Sync:** API integration to fetch and update live schedules and repair tickets instantly.
* **🎨 Premium UI/UX:** Built with Tailwind CSS, featuring glassmorphism, animated scanlines, and custom gradients (Slate, Cyan, Purple).

## 🚀 Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App/Pages Router)
* **Library:** React 18
* **Styling:** Tailwind CSS
* **Database:** MongoDB
* **Icons/Emojis:** Native OS Emojis
